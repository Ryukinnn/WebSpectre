# Changelog

Semua perubahan yang signifikan pada proyek **WebSpectre** akan didokumentasikan di file ini.

Format dokumen ini mengikuti prinsip [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), dan proyek ini mematuhi [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-15
### Ditambahkan
- Inisialisasi awal repositori proyek WebSpectre.
- Skrip instalasi (`setup.py`) untuk memvalidasi kompatibilitas Python 3.12+ dan OS berbasis Linux.
- Antarmuka CLI sentral (`spectre.py`) menggunakan library `rich` untuk pengalaman terminal premium.
- Struktur direktori modular yang memisahkan *core*, *modules*, *plugins*, dan komponen pelaporan.
- Arsitektur berbasis *Clean Architecture* tanpa kode referensi *(placeholder)* atau hasil tiruan.
- Berkas dokumentasi utama: `README.md`, `LICENSE`, `.gitignore`, dan konfigurasi `pyproject.toml`.
- Standarisasi output bahasa Indonesia untuk seluruh UI/UX.
